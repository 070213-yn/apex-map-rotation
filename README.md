# Apex ランクマップローテーション早見表

Apex Legends のランクマップローテーションを、現在時刻から3日後まで自動表示するシンプルなWebサイトです。

## 仕様

- **ローテーション周期**: 4時間30分ごと
- **マップ順**: ブロークンムーン → キングスキャニオン → オリンパス
- **起点**: 2026-05-11 17:00 JST = ブロークンムーン
- **タイムゾーン**: 日本時間(JST)固定で表示
- カウントダウンは1秒ごとに更新、スロット切替時に全体再描画

## ファイル構成

```
.
├── index.html       … マークアップ
├── styles.css       … Apex風ダークデザイン
├── app.js           … ローテ計算 & 描画
├── images/          … 各マップのサムネ画像 (約100KB/枚)
└── README.md
```

## ローカルで開く

`index.html` をブラウザでダブルクリックするだけで動作します。

## デプロイ

GitHub Pages で配信。ブランチは `main`、ディレクトリは `/`。
